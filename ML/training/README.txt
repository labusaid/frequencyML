Directory containing notebooks related to training of models.

File list: 
  - first_ml_model.ipynb
  - ml_batch_trainer.ipynb
  - training_script.ipynb
  
 first_ml_model.ipynb:
    Prototype of the training and testing of the first ML model made.
    
 ml_batch_trainer.ipynb:
    Notebook dedicated to testing the effects of different variables on the accuracy and training of ML models.
    Structured into function and main blocks, with the main blocks controlling the flow of the program.
    
 training_script.ipynb:
    Python script that takes a model in SavedModel format and a hdf5 file containing training data, trains the model with the training data, 
    and outputs the newly trained model in SavedModel format
